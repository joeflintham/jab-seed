# jab-seed

This is a seed repository for projects using jquery-angular-bootstrap.

## Requirements

- Node / npm
- Ruby / rubygems

## Installation

```bash
    gem install sass
```

```bash
    npm install
```

## Development server

```bash
   npm run dev
```

## Tests

TDD:

```bash
   npm run test
```

e2e:

```bash
   npm run e2e
```
